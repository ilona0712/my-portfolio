# Personal Portfolio — Ilona Chamoun

A personal portfolio website built with React, TypeScript, and Vite. Designed to showcase my projects, technical skills, and professional background as a Software Engineering student at USJ Beirut.

## Overview

This portfolio was designed from a custom Figma mockup and implemented as a fully responsive single-page application. It serves as a central hub linking my work, skills, and contact information for recruiters and collaborators.

---

## Sections

- **Hero** — Introduction and headline
- **About** — Background, education, and current focus
- **Projects** — Highlighted work including the NUMU Ministry Dashboard (Code for Lebanon Top 3 Finalist) and the Embedded Sound Tracking System
- **Skills** — Technical stack: languages, frameworks, tools
- **Experience** — Internship timeline at Bank Byblos and DigitalCircle
- **Contact** — Links and contact form

## Getting Started

### Prerequisites

- Node.js v18+
- npm

### Installation

```bash
git clone https://github.com/ilona0712/my-portfolio.git
cd my-portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for production

```bash
npm run build
npm run preview
```
